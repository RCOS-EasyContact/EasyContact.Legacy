# Install Guide
This Guide Will Help You Install Easy Contact

## Required Softwares
- Frontend
  - Nodejs
- Backend
  - Make
  - Clang

## Compile
Clone the Repository:
```shell
git clone https://github.com/RCOS-EasyContact/EasyContact.git
cd EasyContact/
```
Update Sub-Modules:
```shell
git submodule update --init --recursive
```
Make Backend:
```shell
cd Backend/
make
```
